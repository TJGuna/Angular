Introduction to Data Binding in Angular

Data Binding in Angular is a mechanism that allows communication between the component and the view (HTML). It enables dynamic interaction where changes in the component automatically reflect in the view and vice versa. Angular provides multiple types of binding to achieve this, allowing for powerful and flexible UI development.

Types of Data Binding in Angular:

2. Property Binding ([property]="value")

Binds a property of an HTML element to a component property.

<img [src]="imageSrc">

3. Event Binding ((event)="method()")

Binds an event, such as a button click, to a component method.

<button (click)="onClick()">Click</button>

4. Two-Way Binding ([(ngModel)]="property")

Binds component data to a form control and vice versa.

<input [(ngModel)]="name">

5. Class Binding ([class.className]="condition")

Dynamically adds or removes CSS classes based on component data.

<div [class.active]="isActive"></div>

6. Style Binding ([style.property]="value")

Binds inline styles of an HTML element to component data.

<div [style.background-color]="bgColor"></div>

7. Attribute Binding ([attr.attribute]="value")

Binds an attribute of an HTML element to a component property.

<button [attr.aria-label]="buttonLabel">Click</button>
